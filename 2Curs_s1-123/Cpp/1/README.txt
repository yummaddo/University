program.exe - з компонований файл готовий для використання як незалежний елемент

В програмі використовується скомпільований обєкт sqlite3.o для використання можливостей sqlite3

main.cpp - основний код програми 
main.o - скомпонований обєкт main.cpp
sqlite3.c - виконуючий файл бібліотеки sqlite3
sqlite.h - файл заголовків для бібліотеки sqlite3

Для того щоб самостійно скомпонувати новий файл program.exe
потрібно запустити компілятор g++ або будь який інший з параметрами sqlite3.o main.o -o program.exe

При внесенні змін в main.cpp та перевірки його роботоспособності та простого тестування 
потрібно запустити компілятор (рекомендовано g++) з параметрами -c main.cpp -o main.o 

run.but - файл для автоматизації процесу компоновки компіляції та старту програми
він запускає програму в новій сесії cmd консолі
